### Passo 1

1. Vá em **Credenciais** e clique no botão **Novas credenciais** para gerar novos parâmetros.
2. Clique no ícone das credenciais que deseja usar e copie o parâmetro: `CLIENT_ID`.
3. Gere suas credenciais acessando o site da plataforma ([gerar credenciais](https://www.youtube.com/)).

> **Importante:** No navegador do usuário, uma URL da Hotmart será aberta para perguntar se ele autoriza a utilização daqueles dados. Caso o usuário aceite, ele será redirecionado de volta para sua aplicação. A **URL de redirecionamento** deve ser informada no parâmetro `REDIRECT_URI`.

---

### Passo 2

1. A URL abrirá uma tela solicitando **Usuário/Senha** do usuário.
2. Após o preenchimento correto, o usuário será redirecionado para uma tela pedindo o consentimento para que o aplicativo faça requisições em seu nome.

| Parâmetro     | Descrição                                        |
| ------------- | ------------------------------------------------ |
| `CODE`        | Obtido no último passo da autenticação            |
| `REDIRECT_URI`| URL utilizada no último passo da autenticação     |
