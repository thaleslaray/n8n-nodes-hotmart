### Hotmart Club API v1 – Endpoint para Listar Páginas de um Módulo

Você pode listar todas as páginas criadas dentro de um determinado módulo na área de membros, usando o endpoint:

```
curl --location --request GET 'https://developers.hotmart.com/club/api/v1/modules/{module_id}/pages' \
	--header 'Content-Type: application/json' \
	--header 'Authorization: Bearer :access_token'
```

#### Parâmetros da Requisição

- **Query:**
  - `subdomain` (string, obrigatório): O subdomínio do Club, definido na administração da área de membros.
- **Path:**
  - `module_id` (string, obrigatório): Identificador único do módulo (consulte o endpoint de módulos para obter).

---

### Tipos de Página

| Tipo          | Definição                                 |
|---------------|------------------------------------------|
| CONTENT       | Página de conteúdo                       |
| ADVERTISEMENT | Página de anúncio                        |
| QUIZ          | Página de perguntas e respostas          |
| WEBINAR       | Página de Webinário                      |

### Tipo de Liberação

| Status   | Definição                                                                      |
|----------|-------------------------------------------------------------------------------|
| BY_DATE  | Página liberada em uma data fixa, definida pelo produtor                      |
| BY_DAYS  | Página liberada após X dias (número de dias definido pelo produtor)           |

---

### Retorno da API
Cada página terá a seguinte estrutura de retorno:

- `page_id` (string): Identificador único da página.
- `name` (string): Nome da página definida pelo produtor.
- `page_order` (integer): Sequência da página dentro do módulo.
- `type` (string): Tipo da página (`CONTENT`, `ADVERTISEMENT`, `QUIZ`, `WEBINAR`).
- `is_published` (boolean): Indica se a página está publicada (`true` ou `false`).
- `total_comments` (long): Total de comentários na página.
- `rates_average` (double): Média das avaliações dos usuários.
- `rates` (object): Média de avaliações detalhada (por nota).
- `has_media` (boolean): Se a página possui mídias relacionadas.
- `liberation_type` (string): Tipo de liberação (`BY_DAYS` ou `BY_DATE`). Ausente em páginas já disponíveis.
- `liberation_date` (long): Data de liberação (`BY_DATE`).
- `liberation_days` (integer): Dias para liberação (`BY_DAYS`).
- `has_duration` (boolean): Se a página tem duração configurada.
- `days_of_duration` (integer): Dias de duração que a página ficará disponível.

**Observações Importantes:**
