### Excluir Cupom

Este endpoint permite excluir um cupom específico do produto na Hotmart. Para isso, é necessário fornecer o identificador único (ID) do cupom que se deseja excluir.

#### Parâmetros da requisição

**Path:**
- `coupon_id` (integer, obrigatório): Identificador único (ID) do cupom que será excluído.

#### Endpoint

```
DELETE /products/api/v1/coupon/:coupon_id
```

#### Exemplo de requisição cURL

```bash
curl --location --request DELETE 'https://developers.hotmart.com/products/api/v1/coupon/:coupon_id' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer :access_token'
```

#### Retorno

Se a exclusão for bem-sucedida, a resposta será:

```
200 OK
{}
```

Nenhum conteúdo é retornado no corpo da resposta (`{}` vazio).

**Atenção:** É obrigatório informar corretamente o `coupon_id` na URL. A autenticação deve ser realizada usando o token de acesso (`Authorization: Bearer :access_token`).
