Aqui estão as informações técnicas completas sobre a notificação de conclusão de módulo via Hotmart (evento CLUB_MODULE_COMPLETED), exatamente conforme a documentação:

## Parâmetros Recebidos (Formato: JSON)

| Parâmetro            | Descrição                                                                                                                                                                                                                   |
|----------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **hottok** (string)  | Cada conta possui um token único. Ele é a principal garantia de que a requisição está sendo feita pela Hotmart, protegendo contra fraudes e ataques. Este campo é enviado como `X-HOTMART-HOTTOK` no cabeçalho HTTP e deve ser validado. |
| **id** (string)                | Código único de identificação do evento recebido.                            |
| **creation_date** (long)         | Data de criação do evento, em milissegundos desde 1970-01-01 00:00:00 UTC (**Unix Epoch**).           |
| **event** (string)             | Nome do evento: sempre será `CLUB_MODULE_COMPLETED`.                                               |
| **version** (string)           | Versão do evento: sempre será "2.0.0".                                                            |
| **data** (object)            | Dados relacionados ao evento. Veja detalhes abaixo.                              |
| data.**product** (object)   | Informações sobre o produto.                                                                       |
| data.product.**id** (integer)     | Identificador único do produto.                                                                     |
| data.product.**name** (string)   | Nome do produto.                                                                                   |
| data.**user** (object)      | Informações sobre o aluno.                                                                         |
| data.user.**name** (string)      | Nome completo do aluno.                                                                            |
| data.user.**email** (string)     | E-mail do aluno.                                                                                   |
| data.**module** (object)    | Informações sobre o módulo.                                                                         |
| data.module.**id** (string)      | Identificador único do módulo.                                                                     |
| data.module.**name** (string)    | Nome atual do módulo.                                                                              |

---


## Observações Importantes
- Para alunos consumindo via mobile, a notificação poderá depender da versão do Hotmart App instalada.
- Sempre valide o token enviado em `X-HOTMART-HOTTOK` no header HTTP antes de tratar os dados recebidos.

Se precisar alterar a chave do token, é necessário entrar em contato com o suporte Hotmart.

Caso precise de mais detalhes ou exemplos de uso, consulte os tutoriais de webhook Hotmart.
