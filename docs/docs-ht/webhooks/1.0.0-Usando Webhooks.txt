## Usando Webhooks na Hotmart

A ferramenta de notificações da Hotmart permite integrar Webhooks a qualquer sistema para automatizar o controle dos seus produtos. 

### O que são Webhooks?
Webhooks são usados para notificar o seu sistema sempre que um evento acontecer em seus produtos, como:
- Pagamento confirmado
- Contestação de cobrança
- Pagamento recorrente bem-sucedido

Você pode escolher quais eventos deseja receber via Webhook.

### Como realizar a integração via Webhooks (Passo a passo)
1. **Criar um endpoint** ou local para receber as informações.
2. **Configurar** qual é o produto, os eventos e o endereço de recebimento.
3. **Testar a integração**.

### Como começar
- Crie sua configuração e comece a receber eventos agora mesmo pelo painel Hotmart: [Criar uma configuração](https://app-vlc.hotmart.com/tools/webhook)

### Boas práticas
- Aprenda boas práticas de uso de Webhooks e APIs para otimizar seus processos, economizando no processamento, tratamento e recuperação de dados.
